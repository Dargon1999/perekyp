# Техническая документация MoneyTracker

## Общая архитектура
Приложение построено на базе **PyQt6** с использованием паттерна асинхронной обработки данных для обеспечения отзывчивости UI.

### Ключевые компоненты
- **MainWindow**: Главный контейнер, управляющий навигацией и системными треями.
- **AsyncDataManager**: Прослойка для выполнения тяжелых операций с данными в фоновых потоках.
- **HotkeyManager**: Модуль глобального перехвата клавиш (F7/F8) через библиотеку `keyboard`.
- **StyleManager**: Централизованное управление темами (Dark/Light/Dark Blue) и QSS-стилями.

## Производительность и стабильность
- **Lazy Loading**: Вкладки инициализируются только при первом обращении или с небольшой задержкой после старта.
- **UI Heartbeat**: Встроенная система мониторинга (`PerformanceMonitor`) отслеживает зависания главного потока.
- **Логирование**: В продакшене используется уровень `ERROR`. Для отладки установите `DEBUG_MODE=1`.

## Горячие клавиши
- **F7**: Открывает интерфейс инструментов очистки.
- **F8**: Запускает мгновенную фоновую очистку временных файлов и кэша.

# Руководство пользователя

## Начало работы
1. Запустите `main.py`.
2. При первом запуске приложение может запросить права администратора для корректной работы инструментов очистки.

## Основные функции
- **Очистка (F7/F8)**: Используйте F8 для быстрой очистки системы без отвлечения от работы.
- **Настройки**: Здесь вы можете изменить тему оформления, цвет акцента и настроить уведомления.
- **Автоматизация**: Приложение автоматически сохраняет ваши данные и положение окна.

## Безопасность
Все операции очистки проходят через фильтр безопасности, исключающий удаление критически важных системных файлов Windows.
