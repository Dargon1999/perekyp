# MoneyTracker V9.2 - Документация

Система управления капиталом и перепродажами.

## 🚀 Основные возможности
- **Единый баланс**: Все операции во всех вкладках (Рыбалка, Перекуп авто, Майнинг и т.д.) автоматически влияют на общий баланс профиля.
- **Точность вычислений**: Использование системы минорных единиц (копейки/центы) для исключения ошибок округления.
- **Рыбалка 2.0**: Встроенный калькулятор снастей, поиск по операциям и групповое добавление товаров.
- **Планирование**: Установка целей и автоматический прогноз их достижения.

## 📖 Руководство пользователя

### 1. Управление капиталом
Во вкладке **«Капитал»** отображается ваш текущий ликвидный баланс (Cash) и общая стоимость активов (Net Worth).
- **Ликвидный баланс**: Реальные деньги, которые у вас есть.
- **Net Worth**: Ликвидные деньги + стоимость товаров в инвентаре (одежда, авто).

### 2. Вкладка «Рыбалка» (Обновлено)
- **Подбор снастей**: Многофильтровый поиск по названию, описанию, диапазону цен и первой букве. Мгновенная фильтрация с индикацией активных фильтров.
- **Регионы и рыба**: Адаптивная сетка карточек регионов с тултипами и описанием улова.
- **Всплывающий калькулятор**: Плавающее окно с поддержкой стандартных операций (+, -, *, /, %). Можно перетаскивать и сохранять позицию.
- **База рыб**: CRUD-раздел для видов рыб с поддержкой фото, описаний и системы трофеев.
- **Быстрая продажа**: Специальный блок для мгновенного добавления дохода от пойманной рыбы прямо из базы трофеев.
- **Селект снастей**: Выпадающий список с поиском и поддержкой 1000+ элементов.

#### Горячие клавиши калькулятора:
- `C` - Очистить
- `⌫` (Backspace) - Удалить последний символ
- `=` (Enter) - Посчитать результат
- `%` - Процент от числа
- `.` - Десятичный разделитель

## 🛠 API Документация (DataManager)

### Класс `Money` (utils.py)
Используется для всех денежных операций. Хранит значение в целых числах (минорные единицы).
- `Money.from_major(float_amount)`: создание из обычного числа.
- `money.to_major()`: получение float значения.

### Основные методы `DataManager`:
- `get_total_capital_balance()`: возвращает словарь с `liquid_cash` и `net_worth`.
- `get_category_stats(category)`: возвращает статистику по конкретной вкладке (доход, расход, чистая прибыль).
- `add_transaction(category, amount, comment, date, item_name, ...)`: добавляет операцию. Автоматически обновляет глобальный баланс.

### Миграция данных
При обновлении до V9.2 система автоматически:
1. Удаляет устаревшие поля `starting_amount` из всех категорий.
2. Переносит управление балансом на уровень профиля.
3. Добавляет недостающие категории (например, `fishing`).
