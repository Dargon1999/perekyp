# MoneyTracker - Готовые файлы для установки

## 📁 Структура сборки

```
build_output/
├── MoneyTracker_Windows.exe    ← EXE файл для Windows (28.5 MB)
├── MoneyTracker_PWA.zip        ← PWA пакет для всех платформ
├── PWA/                        ← Распакованный PWA пакет
│   ├── web/                    ← Веб-файлы приложения
│   ├── run_server.py           ← Запуск сервера
│   ├── start_server.bat        ← Старт для Windows
│   ├── start_server.sh         ← Старт для Linux/Mac
│   └── README.md               ← Инструкция
└── Android/                    ← Файлы для Android APK
    ├── twa-manifest.json       ← Конфигурация TWA
    └── README.md               ← Инструкция по созданию APK
```

---

## 🖥️ WINDOWS (ПК)

### Быстрый запуск:
1. Скачайте `MoneyTracker_Windows.exe`
2. Запустите двойным кликом
3. Приложение откроется в браузере

### Альтернативный способ:
1. Распакуйте `MoneyTracker_PWA.zip`
2. Запустите `start_server.bat`
3. Откройте http://localhost:5000 в браузере

---

## 📱 ANDROID (Телефон/Планшет)

### Способ 1: PWA установка (Рекомендуется)
1. Запустите сервер на ПК (через EXE или start_server.bat)
2. Узнайте IP-адрес ПК: `ipconfig` в командной строке
3. На телефоне откройте Chrome
4. Перейдите по адресу: `http://ВАШ_IP:5000`
5. Нажмите "Установить приложение"

### Способ 2: Создание APK (Продвинутый)
1. Перейдите на https://www.pwabuilder.com/
2. Введите URL вашего сервера
3. Выберите "Android"
4. Скачайте APK
5. Установите на телефон

### Способ 3: Capacitor (Для разработчиков)
```bash
cd mobile_build
npm install
npx cap add android
npx cap sync
npx cap open android
```

## 🍎 iOS (iPhone/iPad)

### PWA установка (Единственный способ для PWA):
1. Запустите сервер на ПК
2. Узнайте IP-адрес ПК
3. На iPhone/iPad откройте **Safari** (обязательно!)
4. Перейдите по адресу: `http://ВАШ_IP:5000`
5. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
6. Выберите "На экран «Домой»"
7. Нажмите "Добавить"

**Важно:** На iOS PWA работает только через Safari!

## 🔄 СИНХРОНИЗАЦИЯ ДАННЫХ

### Как синхронизировать между ПК и телефоном:

1. **Запустите сервер** на ПК
2. **На ПК**: Откройте http://localhost:5000/sync
3. Нажмите "Экспорт данных" → сохраните JSON файл
4. **На телефоне**: Откройте http://ВАШ_IP:5000/sync
5. Нажмите "Импорт данных" → выберите JSON файл

### Автоматическая синхронизация:
- Откройте страницу синхронизации на обоих устройствах
- Используйте QR-код для быстрого подключения
- Данные сохраняются на сервере

## 🔧 ТЕХНИЧЕСКАЯ ИНФОРМАЦИЯ

### Требования:
- **Windows**: Windows 10 или выше
- **Android**: Android 5.0 или выше
- **iOS**: iOS 11.3 или выше
- **Python**: 3.10+ (если запускать из исходников)

### Порты:
- Сервер запускается на порту `5000`
- Для изменения измените `run_server.py`

### Файлы данных:
- Данные хранятся в: `~/MoneyTracker/`
- Логи: `~/MoneyTracker/logs/`
- Резервные копии: `~/MoneyTracker/backups/`

## 🚀 БЫСТРЫЙ СТАРТ

### Windows:
```batch
MoneyTracker_Windows.exe
# Или
start_server.bat
```

### Linux/Mac:
```bash
chmod +x start_server.sh
./start_server.sh
# Или
python3 run_server.py
```

### Все платформы:
```bash
pip install -r requirements.txt
python run_server.py
# Откройте http://localhost:5000
```

## 📞 ПОДДЕРЖКА

При возникновении проблем:
1. Проверьте логи в `~/MoneyTracker/logs/`
2. Убедитесь, что порт 5000 не занят другим приложением
3. Проверьте firewall настройки
4. Перезапустите сервер
