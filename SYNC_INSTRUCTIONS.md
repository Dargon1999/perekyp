# Как синхронизировать данные между ПК и телефоном

## Метод 1: Экспорт/Импорт через файл (Рекомендуется)

### Шаг 1: Экспорт данных с ПК
1. Откройте браузер на ПК и перейдите на `http://localhost:5000/sync`
2. Войдите в свою учетную запись
3. Нажмите кнопку **"Экспорт данных на ПК"**
4. Сохраните файл `moneytracker_export_YYYYMMDD.json`

### Шаг 2: Перенос файла на телефон
Выберите удобный способ:
- **Облако**: Загрузите файл в Google Drive, iCloud, Dropbox
- **Email**: Отправьте файл на свой email
- **USB**: Подключите телефон к ПК и скопируйте файл
- **Мессенджер**: Отправьте файл через Telegram, WhatsApp

### Шаг 3: Импорт данных на телефон
1. Откройте браузер на телефоне
2. Перейдите на `http://<IP-адрес-ПК>:5000/sync`
3. Войдите в ту же учетную запись
4. Нажмите **"Импорт данных с телефона"**
5. Выберите сохраненный файл
6. Данные загрузятся на сервер

## Метод 2: Автоматическая синхронизация через сеть

### Настройка:
1. **На ПК**: Узнайте IP-адрес вашей сети
   - Windows: `ipconfig` в командной строке
   - Linux/Mac: `ifconfig` или `ip a`
   
2. **Запустите сервер** с доступом из сети:
   ```bash
   python run_server.py
   ```
   Сервер уже запускается на `0.0.0.0:5000`

3. **На телефоне**: Откройте браузер и перейдите на:
   ```
   http://<IP-адрес-ПК>:5000/sync
   ```
   Например: `http://192.168.1.100:5000/sync`

4. **Автоматическая синхронизация**:
   - На странице синхронизации есть QR-код
   - Отсканируйте его на телефоне
   - Или введите адрес вручную

## Метод 3: Облачная база данных (Для продвинутых)

### Использование PostgreSQL:
1. Создайте базу данных на хостинге (Heroku, Railway, Supabase)
2. Измените `DATABASE_URL` в `.env`:
   ```
   DATABASE_URL=postgresql://user:password@host:5432/dbname
   ```
3. Перезапустите сервер
4. Теперь данные хранятся в облаке и доступны везде

### Использование Firebase:
1. Создайте проект в Firebase
2. Включите Firestore Database
3. Добавьте Firebase SDK в проект
4. Настройте синхронизацию через Firebase

## Решение проблем

### Не удается подключиться с телефона:
1. Убедитесь, что телефон и ПК в одной Wi-Fi сети
2. Проверьте, не блокирует ли防火墙 (Firewall) порт 5000
3. Попробуйте отключить антивирус временно

### Файл импорта не принимается:
1. Проверьте, что файл имеет расширение `.json`
2. Убедитесь, что файл не поврежден
3. Попробуйте экспортировать заново

### Данные не синхронизируются:
1. Проверьте интернет-соединение
2. Перезапустите сервер
3. Очистите кэш браузера

## Безопасность

### Рекомендации:
1. **Используйте HTTPS** для продакшена
2. **Меняйте пароль** регулярно
3. **Не храните** файлы экспорта в общедоступных местах
4. **Используйте VPN** при подключении извне

### Защита данных:
- Все данные шифруются при передаче (HTTPS)
- Пароли хешируются
- Регулярно делайте бэкапы

## Автоматическое резервное копирование

### Настройка cron (Linux):
```bash
# Ежедневный бэкап в 3:00
0 3 * * * cd /opt/moneytracker && python -c "from web import create_app, db; app=create_app(); with app.app_context(): # логика бэкапа"
```

### Настройка Task Scheduler (Windows):
1. Откройте Планировщик задач
2. Создайте новую задачу
3. Укажите путь к Python и скрипту бэкапа
4. Установите расписание
