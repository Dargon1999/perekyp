Mem Reduct Pro - Горячие клавиши F7 и F8
==========================================

ОПИСАНИЕ:
-----------
F7 - Запуск Mem Reduct Pro
    * Если программа уже запущена (даже если свернута в трей) - разворачивает окно
    * Если программа не запущена - запускает её
    
F8 - Быстрая очистка TEMP
    * Мгновенно запускает очистку временных файлов
    * Показывает уведомление: "TEMP: освобождено X.X МБ/ГБ, файлов: N"

УСТАНОВКА:
-----------
1. Убедитесь, что файлы находятся в папке с Mem Reduct Pro:
   - mem_reduct_hotkeys.py (слушатель горячих клавиш)
   - temp_cleaner.py (модуль очистки TEMP)

2. Запустите mem_reduct_hotkeys.py:
   python mem_reduct_hotkeys.py
   
   Или добавьте в автозагрузку Windows для постоянной работы.

ИСПОЛЬЗОВАНИЕ:
--------------
1. Запустите mem_reduct_hotkeys.py (он будет работать в фоновом режиме)
2. Нажмите F7 - откроется Mem Reduct Pro (или активируется, если уже запущен)
3. Нажмите F8 - произойдет очистка TEMP с уведомлением

ПРИМЕР ВЫВОДА ДЛЯ F8:
--------------------
TEMP: освобождено 15.2 МБ, файлов: 23

ТРЕБОВАНИЯ:
-------------
* Windows 7/8/10/11
* Python 3.6+ (с библиотеками ctypes, os, subprocess)
* Права администратора (рекомендуется для F8)

ПРИМЕЧАНИЕ:
------------
* Для корректной работы F7, убедитесь что:
  - mem_reduct_pro.exe или mem_reduct_pro.py находится в той же папке
  - или укажите правильный путь в файле mem_reduct_hotkeys.py
* Уведомления работают через PowerShell (Windows 10+)
